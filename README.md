## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/tusharbeshra17-glitch/python-quiz-game.git
```

2. Open the project folder

```bash
cd python-quiz-game
```

3. Run the program

```bash
python quiz_game.py
```
